### Clean Flash
```
- Download the latest build
- Take a backup for safe side
- Boot to Recovery - must use A14 recovery - link below
- Flash the latest build
- Format Data
- Reboot
```

### Dirty Flash
```
- Reboot to Recovery
- Flash ROM zip
- Wipe cache and dalvik
- Reboot and voila!

```
> [!Important]
